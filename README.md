# Personal Portfolio | Gregorio Viola

Personal Portfolio project developed for **Start2Impact** Fullstack Development Master.

## Demo
Find the live project at: [tiredviking.github.io](https://tiredviking.github.io)

## Tech Stack
* **HTML5**: with SEO optimization.
* **CSS3 / Sass (SCSS)**
* **Bootstrap 5**

## Project structure
```text
.
├── README.md
├── node_modules                 # npm package for Bootstrap
├── cv.html                      # CV page (EN)  
├── cv_ita.html                  # CV page (IT)  
├── index.html                   # Main page, entry point (EN)  
├── main-it.html                 # Main page (IT)
└── assets/  
    ├── img/                     # Favicons  
    └── styles/  
         └── sass/               # Source SCSS  
              ├── main.scss      # main .scss file  
              ├── cv.scss        # scss file for cv pages  
              ├── namespaces/    # namespaces imported by main.scss and cv.scss
              ├── purged_css/    # contains the purged version of compiled scss files; these are the ones used in the final html files
              └── compiled_sass/ # css files compiled from main.scss and cv.scss
```
